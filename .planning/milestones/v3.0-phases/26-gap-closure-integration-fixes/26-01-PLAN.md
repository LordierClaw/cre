    Add the following static methods to AstUtils:
    1. `normalizeType(String typeName)`: Strips generics and FQN prefixes (e.g., "java.util.List<String>" -> "List").
    2. `getMethodSignature(MethodDeclaration md)`: Generates "name(param1,param2)" using normalized type names.
    3. `getConstructorSignature(ConstructorDeclaration cd)`: Generates "name(param1,param2)" using normalized type names.
    4. `getResolvedMethodSignature(ResolvedMethodDeclaration rmd)`: Generates "declaringFqn::name(param1,param2)" using normalized type names.
    5. `getResolvedConstructorSignature(ResolvedConstructorDeclaration rcd)`: Generates "declaringFqn::name(param1,param2)" using normalized type names.
  </action>
